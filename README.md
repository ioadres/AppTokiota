# AppTokiota (Dev) 

## Supported Platforms: iOS, and Android

The AppTokiota app is available for two platforms:

- iOS
- Android

## Xamarin.Forms App (AppTokiota)

- Xamarin.Forms
- XAML UI
- Converters
- Custom Controls
- Custom Renderers
- Data Binding
- Effects
- Behaviors
- Custom Markup Extensions
- IoC
- Messaging Center
- MVVM
- Plugins for Xamarin
- Styles
- Phone and Tablet Layout
- Xamarin UITests
